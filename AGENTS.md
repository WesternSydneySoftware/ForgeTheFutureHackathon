# EnRoute (Forge The Future Hackathon)

## Team
- Name: Matthodical

## Problem Statement
Tradies often travel home with unused time, tools, and skills, while nearby customers have small jobs that remain unfinished because they are too minor to book through traditional services. There is no efficient system that matches tradies with quick, nearby tasks along their existing commute. This results in lost income opportunities for tradies and delayed solutions for customers.

## Solution
Build a real-time micro-job platform that suggests small, nearby jobs to tradies based on their live location and commute route. Tradies can instantly accept jobs that match their skills and equipment, allowing them to earn extra income with minimal disruption. Customers benefit from faster, cheaper job completion by connecting with nearby available tradies.

## Tech Stack
- **Node.js (Express):** backend API + static file host.
- **Elastic Cloud (Elasticsearch):** core search + datastore for jobs; fast geospatial queries for nearby matching.
- **Frontend (HTML/CSS-first):** static HTML + CSS with vanilla JS for calling the API (minimal JavaScript only where necessary).
